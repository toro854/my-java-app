# フラッシュ暗算系のゲーム

## デプロイURL
https://my-java-app-q9xd.onrender.com/

## 使用技術
- Java（メインロジック）
- JavaScript（フロント制御）
- HTML / CSS（UI構築）
※Spring Bootは未使用（今後導入予定）

## ゲーム機能
### ■ モード
- 暗算モード（表示された数字の合計を回答）
- 記憶モード（表示された数字の順番を回答）

### ■ 難易度
- 複数レベル（表示速度・難易度が変化）

### ■ スコア機能
- チャレンジモードでスコア計測機能あり

## 工夫した点
- ゲームモードを分けて、異なる認知能力（計算力・記憶力）を鍛えられる設計にした
- レベル制を導入し、難易度調整を可能にした
- 一桁の数字しか出したくなかったので、左右反転や90度傾けて表示などして難易度アップするようにした
- 漢数字などを混ぜることによる難易度アップするようにした
- マウスのみやスマホでのタップ操作ができるように4択式にした
- フロントとロジックを分離して実装

## 今後の改善案
- 現在はクライアント完結型のため、Spring BootによるAPI化を行い、責務分離を進める
- スコアをDBで管理し、永続化およびランキング機能を実装する
- ユーザー単位のログイン機能を追加し、競争性を持たせる
